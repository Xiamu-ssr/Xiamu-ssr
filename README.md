## selected projects

### [berry-agent-sdk](https://github.com/Xiamu-ssr/berry-agent-sdk) 🍓
Pure-library TypeScript agent runtime.
Agent loop, session resume / fork, batch compaction, cache-aware provider adapters, tool calling, streaming events.

### [CCodeBoX](https://github.com/Xiamu-ssr/ccodebox) 📦
Execution environment layer for coding agents.
Git worktree isolation + YAML stage orchestration for code → test → fix → deliver workflows.

### [AgentIM](https://github.com/Xiamu-ssr/AgentIM) 💬
Messaging system for agents.
Self-hosted, CLI-first, real-time messaging, full-text search, Ed25519-based auth.

### [berry-claw](https://github.com/Xiamu-ssr/berry-claw) 🐾
Product built on top of `berry-agent-sdk`.
Used to validate runtime and API design under real usage.
